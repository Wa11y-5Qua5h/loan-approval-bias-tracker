# loan-approval-bias-tracker


## Overview

This is a machine learning and data analysis project designed to identify potential bias in loan approval decisions. The system analyzes loan application datasets and evaluates whether approval outcomes differ significantly across demographic groups.

## Objectives

* Detect patterns of bias in loan approval data.
* Analyze approval rates across demographic categories.
* Provide visualizations and insights into potential disparities.
